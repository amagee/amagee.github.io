Emacs cheat sheet

Jan 6, 2021
1 min read
  • Rename current file: SPC f R (from Spacemacs config)
  • Open config file: SPC f e f (from Spacemacs config probably)
  • Open buffer list: SPC b b
  • Fuzzy file find: SPC p f
  • Reload config (actually restart Emacs but preserve layout): SPC q r
  • Find out what a key is bound to: C-h c <key>
