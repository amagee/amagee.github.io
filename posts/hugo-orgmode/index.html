Writing Hugo in Org

  • Hugo is a site generator. Among other things it has native support for generating websites from collections of Emacs Org Mode files.

  • What about a link to Emacs

  • It’s as simple as putting .org files in the content/ directory of the the generated site. Hugo has built-in Org Mode support.

    • Hugo will create one page per file. It supports a bunch of different file types out of the box.

  • Use Hugo’s syntax for document metadata:

    #+title: Page title
#+subtitle: Subtitle
#+date: 2021-01-10
#+tags[]: tag1, tag2, tag3
